# チャレコ
チャレンジ特化型のSNSです。
チャレンジの投稿や閲覧が可能です。
レスポンシブ対応しているのでスマホからもご確認いただけます。

# URL
https://chalreco-ed297b86e79e.herokuapp.com/

# 使用技術
* PHP 8.0.30
* Laravel 8.83.27
* Nginx 1.20.2
* MySQL 8.0.34
* Docker 20.10.21

# 機能一覧
* ユーザー登録、ログイン機能
* 投稿機能
* 閲覧機能
